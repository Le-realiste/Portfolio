Description du projet

Développement d’un chatbot en Java capable de répondre à des questions enregistrées et d’apprendre de nouvelles réponses grâce à un système d’indexation.

Objectifs

  • Créer un programme interactif en Java
  • Mettre en place un système d’apprentissage
  • Indexer les questions pour accélérer la recherche
  • Gérer les fichiers pour stocker les connaissances

Compétences mobilisées

  • C1 — Réaliser un développement d’application
  • C2 — Optimiser des applications

Technologies utilisées

Java, Algorithmique, Fichiers texte
