Description du projet

Ce projet consistait à créer un site vitrine pour IBM, simulant une demande client réelle. L’objectif était de présenter l’entreprise, ses activités principales et ses valeurs.

Objectifs

  • Créer un site web statique professionnel
  • Structurer une page en HTML
  • Styliser avec un design moderne en CSS
  • Respecter une demande client simulée

Compétences mobilisées

  • C5 — Conduire un projet
