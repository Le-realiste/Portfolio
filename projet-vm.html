Description du projet

Installation et configuration d’une machine virtuelle Debian 13 avec KDE/Plasma. Mise en place des outils essentiels : Git, JDK, IntelliJ IDEA (via archive, snap et flatpak).

Objectifs

  • Installer un système Linux complet
  • Configurer un environnement de développement
  • Comprendre les différences entre Snap, Flatpak et archives
  • Documenter toutes les étapes

Compétences mobilisées

  • C3 — Administrer des systèmes informatiques
  • C5 — Conduire un projet

Technologies utilisées

Debian 13, KDE/Plasma, Git, Bash, IntelliJ IDEA
