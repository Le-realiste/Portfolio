Description du projet

Conception d’une base de données pour une association maritime. Réalisation du MCD, MLD, scripts SQL, requêtes avancées et mises à jour.

Objectifs

  • Analyser les besoins de l’association
  • Créer un MCD et un MLD cohérents
  • Écrire les scripts SQL de création et suppression
  • Réaliser des requêtes complexes (jointures, agrégations…)

Compétences mobilisées

  • C4 — Gérer des données
  • C6 — Travailler en équipe

Technologies utilisées

SQL, PostgreSQL, Modélisation MCD/MLD
